# 新一代物业管理系统 java656

## 项目概述


## 技术栈

### 后端

- **Spring Boot 3**
- **MyBatis**
- **MySQL**
- **Redis**

### 前端

- **Vue 3**
- **Element Plus**

## 功能模块

- **房产管理**：便于管理和维护房产信息。
- **在线缴费**：支持住户在线支付物业费、水电费等。
- **报修服务**：住户可在线提交报修需求。
- **安保管理**：加强社区安保工作，提高居住安全。

## 系统角色

- 物业管理员：管理后台操作人员。
- 业主：通过门户进行操作的用户。

## 运行环境

- **后端**：Java 17+环境，推荐使用IntelliJ IDEA、Eclipse等IDE。
- **前端**：Node.js环境，推荐使用Visual Studio Code等编辑器。

## 部署步骤

1. 准备好运行环境。
2. 导入数据库文件。
3. 部署后端服务。
4. 部署前端服务。
